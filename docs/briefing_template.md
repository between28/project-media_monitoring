# 브리핑 템플릿

현재 브리핑 Markdown은 아래 구조를 사용합니다.

## 1. 우리부 보도자료

- `HWPX` 본문에서 `□`로 시작하는 첫 본문 줄만 그대로 사용
- 추가 요약 문장 자동 생성 없음

예시:

```text
[우리부 보도자료]
국토교통부 ... 현장 관계자를 격려했다.
```

## 2. 언론보도 현황

- 실행 시점 기준 현재 시각을 표기
- 기사 목록과 같은 기준의 관련 기사 건수를 사용
- 대표 기사 1건을 각주 형식으로 표시

예시:

```text
[언론보도 현황]
2026년 3월 13일 14:30 현재, 관련 보도: BBS불교방송, 노컷뉴스, 뉴스1 등 14건*
   * [뉴시스] 대광위원장 "동북선 경전철 개통땐 서울 동북권 교통편의 향상 기대" 등
```

## 3. 참고자료

브리핑 본문과 별도로 기사 목록 파일이 생성됩니다.

출력 파일:
- `outputs/references/D+N_YYYY-MM-DD_기사목록.csv`
- `outputs/references/D+N_YYYY-MM-DD_기사목록.md`

기본 컬럼:
- `순번`
- `언론사`
- `기사 제목`
- `보도일시`
- `기사 링크`

## 작성 원칙

- 작위적인 템플릿 문장은 최소화합니다.
- 기사 수는 참고기사 목록과 같은 기준을 우선합니다.
- 브리핑 본문은 짧고 사실 중심으로 유지합니다.
